# LaTeX Paper Templates

This directory stores optional LaTeX templates for final paper artifact generation.

Each template lives in its own subdirectory and should preserve upstream filenames so template-local includes keep working. Generated papers should copy from these templates into the launch output directory instead of modifying template files in place.

## Available Templates

**iclr2026**

- Source: https://github.com/ICLR/Master-Template
- Upstream commit: `a28d335b0d46a3c39b205704a65faf41c9748433`
- Download method: shallow clone of the official ICLR template repository, copying the `iclr2026/` template directory.
- Primary TeX entrypoint: `iclr2026_conference.tex`
- Style file: `iclr2026_conference.sty`
